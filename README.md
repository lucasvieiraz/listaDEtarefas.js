## Lista de Tarefas

Este projeto é uma página web simples que permite ao usuário adicionar itens dinamicamente a uma lista. Ele foi desenvolvido utilizando HTML, CSS e JavaScript combinados em um único arquivo.

## Funcionalidades
Adicionar itens à lista.
Exibir os itens adicionados dinamicamente.
Interface simples e amigável.

## Tecnologias Utilizadas
HTML5: Estrutura da página.
CSS3: Estilização da interface.
JavaScript: Lógica para adicionar e exibir os itens da lista.
